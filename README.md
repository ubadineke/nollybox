# Nollybox

[![Watch the demo](https://img.youtube.com/vi/u3Z_KNJPMBg/maxresdefault.jpg)](https://youtu.be/u3Z_KNJPMBg)

Nollybox is a fictional Nollywood streaming service built as a demo consumer for **[Plinth](https://github.com/ubadineke/plinth)** — subscription billing infrastructure on top of Nomba's payment rails.

It exists to make Plinth tangible. Plinth is a billing engine — invisible by design. The best way to show what it does is to put a real product on top of it and run the full lifecycle live: a customer picks a plan, pays by card or bank transfer, gets downgraded when payment fails, receives dunning notifications, and recovers — all without Nollybox writing a line of billing logic.

The streaming format is deliberate. Everyone already understands Netflix-style tiers and "your account is on hold." No time is spent explaining the billing model — you just watch it happen.
